# E-Commerce Web Application

A responsive e-commerce web application built using React.js.

## Features
- Product listing and filtering
- Shopping cart management
- User authentication
- Protected routes
- Admin dashboard
- Responsive UI

## Tech Stack
- React.js
- Context API
- JavaScript
- CSS3
- React Router

## Installation

git clone <repo-url>

npm install

npm start
